# 🧠 Quiz Game: Тестирование знаний

Интерактивное веб-приложение для прохождения тестов с визуальной индикацией прогресса, подсчётом баллов и адаптивным дизайном. Проект реализован на чистом JavaScript, HTML и CSS.

## 📚 О проекте

Приложение представляет собой викторину из 6 вопросов на общую эрудицию. После завершения теста пользователь видит свой результат, количество набранных баллов и персональное сообщение в зависимости от процента правильных ответов.

Проект разделён на два логических модуля:
- **module_A** — фронтенд: отображение вопросов, обработка кликов, анимация.
- **module_B** — бэкенд-логика: расчёт баллов, процентов и итогового сообщения.

## 🚀 Инструкция по запуску

1. Запуск приложения
Откройте файл module_A/index.html в любом современном браузере (Google Chrome, Firefox, Edge, Safari).
# Пример для Windows
start module_A/index.html

# Пример для macOS
open module_A/index.html

# Или просто откройте файл двойным щелчком через проводник

2. Запуск unit-тестов (проверка модуля Б)
Для проверки логики подсчёта баллов выполните в терминале:

node test_module_B.js

Ожидаемый вывод:
✅ Модуль Б загружен!
🧪 Тест 1: calculateScore() - пройден
🧪 Тест 2: calculatePercentage() - пройден
🧪 Тест 3: getResultMessage() - пройден
Итого: 5/5 тестов пройдено

🎮 Как играть

1. Нажмите кнопку «Начать тестирование».
2. Читайте вопрос и выбирайте один из четырёх вариантов ответа.
3. После выбора ответ автоматически подсвечивается:
  🟢 Зелёный — правильный ответ.
  🔴 Красный — неправильный ответ.
4. Через 1 секунду загружается следующий вопрос.
5. Вверху отображаются:
  Текущий номер вопроса.
  Счётчик набранных баллов.
  Прогресс-бар заполнения теста.
6. После ответа на все вопросы вы увидите:
  Итоговый балл.
  Персональное сообщение (зависит от процента правильных ответов).
7. Кнопка «Повторить» позволяет пройти тест заново.
