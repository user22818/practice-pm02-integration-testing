# Баг-репорт: Множественные клики засчитываются несколько раз

## Заголовок
[Bug] Быстрые клики по ответам увеличивают счет более чем на 1

## Окружение
- Браузер: Chrome 120+
- ОС: Windows 11

## Шаги воспроизведения
1. Открыть module_A/index.html
2. Начать тест
3. Быстро кликнуть 2 раза по разным ответам

## Фактический результат
Счет увеличивается на 2

## Ожидаемый результат
Счет увеличивается только на 1
